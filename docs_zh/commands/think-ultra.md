# Think Ultra 命令

激活最大认知超思考处理，对以下内容进行超全面分析：**$ARGUMENTS**

## 超分析框架

部署最严格的分析方法，在所有维度上进行详尽检查：

### 阶段 1：问题架构

- **本体分析**：这个问题的根本性质是什么？
- **认识论检查**：我们如何知道我们对这个问题的了解？
- **语义分解**：解构所有关键术语和概念
- **边界分析**：包含什么、排除什么以及为什么？
- **元问题识别**：问题背后的问题是什么？

### 阶段 2：多范式分析

- **还原主义方法**：分解为最小的可分析组件
- **整体系统视图**：检查涌现属性和交互
- **辩证推理**：探索矛盾及其解决
- **现象学视角**：这在主观上如何被体验？
- **实用评估**：实践中什么有效 vs 理论？

### 阶段 3：跨学科整合

- **科学方法**：假设形成、测试、验证
- **数学建模**：定量关系和模式
- **哲学框架**：逻辑一致性和伦理含义
- **历史分析**：模式、先例和进化趋势
- **人类学观点**：文化、社会和行为维度
- **经济分析**：资源配置、激励和权衡

### 阶段 4：时间和空间缩放

- **多时间尺度分析**：即时、短期、中期、长期
- **代际思维**：跨多代的影响
- **空间缩放**：本地、区域、国家、全球影响
- **分形分析**：不同尺度的自相似模式
- **路径依赖**：历史如何约束未来选项

### 阶段 5：不确定性和风险建模

- **概率推理**：贝叶斯更新和置信区间
- **情景规划**：多个未来路径及其影响
- **黑天鹅分析**：低概率、高影响事件
- **反脆弱性评估**：什么从无序中受益？
- **稳健性测试**：各种压力条件下的性能

### 阶段 6：决策理论和博弈论

- **多标准决策分析**：选项的加权评估
- **战略互动**：他人的决策如何影响结果
- **机制设计**：期望结果的最优系统架构
- **行为经济学**：认知偏见和心理因素
- **进化稳定策略**：什么会随时间持续？

### 阶段 7：元认知反思

- **认知偏见审计**：系统识别思维错误
- **视角转换**：强化对立观点
- **假设考古学**：深入挖掘基础信念
- **推理透明度**：使隐含逻辑显式
- **智力谦逊**：承认限制和不确定性

## 超结构化输出

使用此详细格式展示你的全面分析：

### 1. 问题重新概念化

- **原始问题**：如所述
- **精炼问题**：经过深入分析后
- **隐藏假设**：发现的隐含信念
- **重构**：查看问题的替代方式

### 2. 多维度映射

- **核心组件**：基本元素及其关系
- **系统动力学**：反馈循环和涌现行为
- **利益相关者生态系统**：所有受影响方及其利益
- **约束分析**：限制和边界条件

### 3. 证据和研究整合

- **数据综合**：相关的实证发现
- **理论框架**：适用的模型和理论
- **案例研究**：历史先例和类比
- **专家共识**：一致和分歧的领域

### 4. 全面选项分析

- **选项生成**：超越明显选择的创造性替代方案
- **多标准评估**：跨维度的系统比较
- **敏感性分析**：结论对假设变化的稳健性如何？
- **实施路径**：每个选项的实用步骤

### 5. 风险和不确定性评估

- **已知未知**：识别的不确定性领域
- **未知未知**：潜在的盲点
- **故障模式**：什么可能出错以及为什么
- **缓解策略**：风险降低方法

### 6. 战略建议

- **主要建议**：最佳行动方案及理由
- **替代路径**：备用选项和应急方案
- **实施路线图**：带时间线的排序步骤
- **成功指标**：如何衡量进展和结果
- **适应触发**：何时重新考虑方法

### 7. 元分析和反思

- **置信度评估**：你有多确定以及为什么？
- **关键洞察**：最重要的发现
- **剩余问题**：仍需要调查的内容
- **学习机会**：此分析对问题解决的启示

**注意**：此超分析可能需要大量处理时间和计算资源。分析的深度应与问题的复杂性和重要性相匹配。对于不需要完整 7 阶段超全面框架的较不复杂问题，考虑使用 `/think-harder`。
